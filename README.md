# get-signed-32-bit-int

Get a number as a signed 32 bit integer.

## Installation

    $ npm install get-signed-32-bit-int --save

## Usage

    import getSigned32BitInt from "get-signed-32-bit-int";

    getSigned32BitInt(2147483647 + 100) // -2147483549
